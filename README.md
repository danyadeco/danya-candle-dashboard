#  DANYA Candle Dashboard

A Streamlit dashboard for managing candle inventory, sales and profit.

---

##  Version 1.0 - First Release

### Features
- Add Product
- View Product List
- Update Product
- Delete Product
- Record Sales
- Automatic Stock Reduction
- Profit Calculation
- Inventory Tracking
- Business Dashboard Metrics

### Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- Git
- GitHub

---

##  Database Tables

### Products
- ID
- Candle Name
- Sale Price
- Cost Price
- Stock

### Sales
- Product ID
- Quantity Sold
- Sale Date
