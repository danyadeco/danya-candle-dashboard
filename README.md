#  DANYA Candle Dashboard

A Streamlit dashboard for managing candle inventory, sales and profit.

---

##  Version 1.0 - First Release

### Features
- Add Product
- View Product List
- Update Product
- Delete Product
- Record Sales
- Automatic Stock Reduction
- Profit Calculation
- Inventory Tracking
- Business Dashboard Metrics

### Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- Git
- GitHub

---

##  Database Tables

### Products
- ID
- Candle Name
- Sale Price
- Cost Price
- Stock

### Sales
- Product ID
- Quantity Sold
- Sale Date

---
## Version 1.1 - Sales Analytics Update

### Features

- Sale History
  View all recorded sales with product details and sale dates.

- Revenue Dashboard
  Display total revenue, total cost and total profit.

- Best Seller
  Show the best-selling product based on sales quantity.

- Low Stock Alert
  Warn when product inventory becomes low.

- Sales Charts
  Visualize sales performance using charts.
